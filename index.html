3. Eventos são basicamente um conjunto de ações que são realizadas em um determinado elemento da página web.

Alguns dos eventos mais utilizados atualmente são:

Evento "click":

O evento "click" ocorre quando um elemento é clicado pelo usuário com o mouse. É amplamente utilizado em botões, links e elementos interativos para acionar ações específicas.

Evento "submit":

O evento "submit" ocorre quando um formulário HTML é enviado. Geralmente é associado a um botão de envio em um formulário e é usado para processar os dados inseridos pelo usuário no servidor.

Evento "mouseover" e "mouseout":

O evento "mouseover" ocorre quando o cursor do mouse é movido sobre um elemento, enquanto o evento "mouseout" ocorre quando o cursor do mouse é movido para fora do elemento. Esses eventos são usados para criar efeitos de hover, onde as alterações de estilo ou interações ocorrem quando o usuário passa o mouse sobre um elemento.
